# How can you access AWS services?

## 1. AWS Console
- Referred to as **ClickOps** → you click to create and modify resources.

## 2. AWS API
- Directly call AWS APIs.
- Can be done using the **AWS CLI**.

## 3. AWS SDK
- Acts like a wrapper around AWS APIs.
- You can include it inside your code.
- Supports multiple programming languages.
- Useful for automation.

---

# Infrastructure as Code (IaC)
- Build your AWS infrastructure using code (referred to as **DevOps**).
- Write code with all configurations and deploy it to AWS.
- Everything is created at once.

### Example:
- AWS service templates written in **JSON** or **YAML**.
- Most commonly used format: **YAML**
  - You define your architecture in a file.
  - AWS then materializes it into real resources.

---

## Architecture Layers in AWS
- AWS architectures are typically designed in layers:

### Application Layer
- Services like **EC2**, **RDS**, etc.

### Network Layer
- **VPC**
- Subnets
- NAT Gateway
- Route Tables

---

## How CloudFormation Works (Stacks)
- When you write code in **AWS CloudFormation**, it is grouped into something called a **Stack**.
- A **Stack** is a collection of AWS resources created and managed together.

### Key Benefits of Stacks:
- Easily create entire infrastructure in one go.
- Can delete the whole setup cleanly by deleting the stack.
- Supports **stack-to-stack communication**.
- Helps recreate your architecture with a single command (unlike ClickOps).

---

## How to implement IaC

### Tools:
- **AWS CloudFormation**
- **Terraform** (uses `.tf` files)
  - Open-source fork: **OpenTofu** (can use `.yaml`)
- **Pulumi**

### How they work:
- You write infrastructure in your preferred programming language.
- These tools convert it into AWS-compatible formats (like YAML/JSON).
- Then deploy it to AWS.

---

## Notes on YAML
- Not easy to write → **indentation-sensitive**.
- Can be **valid but still incorrect** logically.
- Declarative in nature:
  - Harder to implement loops.
  - Limited flexibility compared to programming languages.
